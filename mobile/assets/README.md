# Assets Directory

This directory should contain the following image files for the Expo app:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

You can create simple placeholder images or download free icons from:
- https://www.flaticon.com/
- https://iconscout.com/
- Or use any image editing tool to create simple colored squares with text

For now, the app will work without these files, but you'll see warnings in the console.
